# librb-http

Non-blocking high-level wrapper for libcurl.

## Example

~~~c
///////////////////////////////////////////////////////////////////////////
//  Uses only the first URL, if it fails then uses the next one
//  2 Max cached connection
///////////////////////////////////////////////////////////////////////////
RBHTTP_HANDLER * handler = rb_http_handler("http://host1,http://host2", 2L);

///////////////////////////////////////////////////////////////////////////
// Enqueue a message (non-blocking)
///////////////////////////////////////////////////////////////////////////
RBHTTP_MESSAGE message;
message.data = "Hello World!";
rb_http_produce(message);
~~~
